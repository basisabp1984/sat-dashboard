# SAT — Демо логістичного дашборду

Жива інтерактивна карта України з 50 вантажівками, потоком вантажів, подіями та сторінкою «Про систему» (3 етапи трансформації логістики).

**Стек:** один `index.html` — Leaflet з OSM-тайлами, без бекенду, без збірки.

## Локальний запуск

Відкрити `index.html` подвійним кліком — браузер покаже карту.

## Деплой

Підключений до Vercel — кожен `git push` оновлює живу версію автоматично.
